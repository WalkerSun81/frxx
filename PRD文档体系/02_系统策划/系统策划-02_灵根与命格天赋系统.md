## 2.1 系统概述

### 系统定位

- **角色初始属性**：在角色创建阶段决定，决定后续成长方向
- **刷初始留存钩子**：通过随机机制提供反复尝试的动力
- **个性化核心**：每个角色拥有独特的属性组合和羁绊效果

### 核心循环

```
随机灵根/命格 → 选择确认 → 激活羁绊 → 影响全游戏数值 → 重置优化（可选）
```

### 与其他系统关系

```
灵根系统 ──────┬──→ 修炼速度倍率
               │
               └──→ 五行属性抗性/伤害
                   
命格天赋 ──────┬──→ 基础属性加成
               │
               ├──→ 被动技能效果
               │
               └──→ 羁绊激活加成
                   
羁绊效果 ──────┬──→ 修炼加成
               │
               ├──→ 属性加成
               │
               └──→ 特殊效果
```

### 设计目标

1. **刷初始留存**：通过随机机制延长玩家在创建阶段的停留时间
2. **角色个性化**：每个玩家拥有独特的成长路径和属性倾向
3. **策略选择**：不同灵根/命格组合适合不同的游戏风格
4. **长期影响**：初始选择对后期发展有显著影响，不可随意更改

### 2.2.3 灵根随机算法

```python
import random

def random_element_root():
    """
    随机生成灵根数据
    返回：elements(五行数值), main(主灵根), cultivate_multiplier(修炼倍率), quality(品质)
    """
    
    # Step 1: 五行各随机1-80的初始值
    elements = {
        'metal': random.randint(1, 80),   # 金
        'wood': random.randint(1, 80),    # 木
        'water': random.randint(1, 80),   # 水
        'fire': random.randint(1, 80),    # 火
        'earth': random.randint(1, 80)    # 土
    }
    
    # Step 2: 找出最大值确定主灵根
    main = max(elements, key=elements.get)
    main_value = elements[main]
    
    # Step 3: 计算修炼加成倍率（基于主灵根）
    if main_value >= 70:
        # 天灵根：天生资质极佳
        cultivate_multiplier = 2.0
        quality = "天灵根"
    elif main_value >= 50:
        # 单灵根：资质优秀
        cultivate_multiplier = 1.7
        quality = "单灵根"
    elif main_value >= 30:
        # 杂灵根：资质普通
        cultivate_multiplier = 1.2
        quality = "杂灵根"
    else:
        # 废灵根：资质极差（但可通过后天努力弥补）
        cultivate_multiplier = 1.0
        quality = "废灵根"
    
    # Step 4: 特殊木灵根额外加成
    if main == 'wood':
        cultivate_multiplier *= 1.1  # 木灵根额外10%修炼加成
    
    return elements, main, cultivate_multiplier, quality


def get_element_bonus(element_type):

def calculate_final_cultivate_speed(base_speed, main_element, element_value):
    """
    计算最终修炼速度
    """
    # 基础修炼倍率
    if element_value >= 70:
        multiplier = 2.0
    elif element_value >= 50:
        multiplier = 1.7
    elif element_value >= 30:
        multiplier = 1.2
    else:
        multiplier = 1.0
    
    # 木灵根特殊加成
    if main_element == 'wood':
        multiplier *= 1.1
    
    return base_speed * multiplier
```

### 2.3.4 命格随机算法

```python
import random

def weighted_random(weights):
    """
    加权随机抽取
    weights: {结果: 权重值}
    返回: 被选中的结果
    """
    total = sum(weights.values())
    rand = random.uniform(0, total)
    cumulative = 0
    for result, weight in weights.items():
        cumulative += weight
        if rand <= cumulative:
            return result
    return list(weights.keys())[-1]


def random_talents():
    """
    随机生成5个天赋
    返回: 天赋列表
    """
    talents = []
    
    for slot in range(5):
        # Step 1: 概率抽取品质
        quality = weighted_random({
            1: 40,  # 凡
            2: 30,  # 良
            3: 15,  # 优
            4: 10,  # 极
            5: 4,   # 仙
            6: 1    # 神
        })
        
        # Step 2: 随机类型
        talent_type = random.randint(1, 6)
        
        # Step 3: 从天赋池中随机选择一个该品质该类型的天赋
        talent_pool = TALENT_POOL[quality]
        type_matched = [t for t in talent_pool if t['type'] == talent_type]
        
        if type_matched:
            talent = random.choice(type_matched)
            talents.append({
                'slot': slot,
                'type': talent_type,
                'quality': quality,
                'name': talent['name'],
                'attr': talent['attr'],
                'value': talent['value'],
                'passive': talent.get('passive', None)
            })
        else:
            # 如果该类型没有匹配的，随机选择一个
            talent = random.choice(talent_pool)
            talents.append({
                'slot': slot,
                'type': talent['type'],
                'quality': quality,
                'name': talent['name'],
                'attr': talent['attr'],
                'value': talent['value'],
                'passive': talent.get('passive', None)
            })
    
    return talents


def calculate_fetters(talents):
    """
    计算激活的羁绊
    返回: 激活的羁绊列表
    """
    activated = []
    qualities = [t['quality'] for t in talents]
    
    # =====================
    # 天道羁绊判定
    # =====================
    
    # 五彩斑斓：5种品质各不相同
    if len(set(qualities)) == 5:
        activated.append({
            'name': '五彩斑斓',
            'category': '天道',
            'effect': '全属性+5%',
            'icon': 'rainbow'
        })
    
    # 五行归一：5种品质相同
    if all(q == qualities[0] for q in qualities):
        activated.append({
            'name': '五行归一',
            'category': '天道',
            'effect': '修炼速度+15%',
            'icon': 'unity'
        })
    
    # 道法自然：有凡(1)有仙(5)或神(6)
    if 1 in qualities and (5 in qualities or 6 in qualities):
        activated.append({
            'name': '道法自然',
            'category': '天道',
            'effect': '修炼速度+10%',
            'icon': 'nature'
        })
    
    # =====================
    # 地道羁绊判定
    # =====================
    
    # 紫气东来：极品以上(4+) >= 3个
    high_quality_count = sum(1 for q in qualities if q >= 4)
    if high_quality_count >= 3:
        activated.append({
            'name': '紫气东来',
            'category': '地道',
            'effect': '气运+20, 悟性+10',
            'icon': 'purple'
        })
    
    # 草根崛起：凡品以下(1-2) >= 3个
    low_quality_count = sum(1 for q in qualities if q <= 2)
    if low_quality_count >= 3:
        activated.append({
            'name': '草根崛起',
            'category': '地道',
            'effect': '气血+10%, 防御+8%',
            'icon': 'grass'
        })
    
    # =====================
    # 人道羁绊判定
    # =====================
    
    # 同舟共济：5个天赋类型各不相同
    types = [t['type'] for t in talents]
    if len(set(types)) == 5:
        activated.append({
            'name': '同舟共济',
            'category': '人道',
            'effect': '所有羁绊效果+50%',
            'icon': 'boat'
        })
    
    # 博而不精：5个天赋类型各不相同，但总值较低
    total_value = sum(t['value'] for t in talents)
    if len(set(types)) == 5 and total_value < 50:
        activated.append({
            'name': '博而不精',
            'category': '人道',
            'effect': '解锁隐藏玩法',
            'icon': 'book'
        })
    
    return activated
```

```python
class ElementRoot:
    """灵根数据模型"""
    
    def __init__(self):
        self.metal = 0      # 金
        self.wood = 0        # 木
        self.water = 0       # 水
        self.fire = 0        # 火
        self.earth = 0        # 土
        self.main_element = None  # 主灵根
        self.quality = None  # 品质
        self.multiplier = 1.0  # 修炼倍率
        self.confirmed = False  # 是否已确认
    
    def confirm(self):
        """
        确认灵根（锁定操作）
        一旦确认，永久不可更改
        """
        if self.confirmed:
            raise ValueError("灵根已确认，无法修改")
        self.confirmed = True
        # 记录确认时间
        self.confirmed_at = datetime.now()
        self.confirmed_at_timestamp = int(time.time())
    
    def is_locked(self):
        """检查灵根是否已锁定"""
        return self.confirmed


class Role:
    """角色数据模型"""
    
    def __init__(self):

        self.talents = []  # 天赋列表
        self.fetters = []  # 激活的羁绊

        """

        if self.element_root.confirmed:
            raise ValueError("灵根已确认，无法随机")
```

```python
def calculate_all_bonuses(role):
    """
    计算角色的所有加成
    这是核心计算函数，在多个场景下被调用
    """
    bonuses = {
        # 基础属性
        '力量': 0,
        '体魄': 0,
        '敏捷': 0,
        '灵力': 0,
        '精准': 0,
        '神识': 0,
        '悟性': 0,
        '气运': 0,
        
        # 战斗属性
        '攻击': 0,
        '防御': 0,
        '气血': 0,
        '暴击率': 0,
        '闪避率': 0,
        
        # 百分比属性
        '受伤减免': 0,
        '攻击加成': 0,
        '修炼速度倍率': 1.0,
        '掉落加成': 0,
        '速度加成': 0,
    }
    
    # =====================
    # 1. 天赋基础属性加成
    # =====================
    for talent in role.talents:
        if talent['attr'] in bonuses:
            bonuses[talent['attr']] += talent['value']
        elif talent['attr'] == '气血':
            bonuses['气血'] += talent['value']
        elif talent['attr'] == '悟性':
            bonuses['悟性'] += talent['value']
    
    # =====================
    # 2. 羁绊加成
    # =====================
    fetters = calculate_fetters(role.talents)
    role.fetters = fetters  # 更新角色羁绊
    
    for fetter in fetters:
        apply_fetter_bonus(bonuses, fetter)
    
    # =====================
    # 3. 灵根加成
    # =====================
    if role.element_root.confirmed:
        # 修炼速度受灵根影响
        element_multiplier = role.element_root.multiplier
        bonuses['修炼速度倍率'] *= element_multiplier
        
        # 主灵根特殊效果
        main = role.element_root.main_element
        if main == 'wood':
            bonuses['修炼速度倍率'] *= 1.05  # 木灵根额外5%
        elif main == 'metal':
            bonuses['精准'] += bonuses['精准'] * 0.1  # 金灵根精准+10%
        elif main == 'fire':
            bonuses['攻击加成'] += 0.1  # 火灵根攻击+10%
        elif main == 'earth':
            bonuses['防御'] += bonuses['防御'] * 0.1  # 土灵根防御+10%
        elif main == 'water':
            # 水灵根气血恢复（此项需特殊处理）
            pass
    
    # =====================
    # 4. 天赋被动技能（特殊处理）
    # =====================
    for talent in role.talents:
        if talent.get('passive'):
            apply_passive_skill(bonuses, talent['passive'])
    
    return bonuses


def apply_fetter_bonus(bonuses, fetter):
    """
    应用羁绊效果
    """
    effect_map = {
        '五彩斑斓': lambda: set_bonus_percentage(bonuses, 'all', 0.05),
        '五行归一': lambda: multiply_bonus(bonuses, '修炼速度倍率', 0.15),
        '道法自然': lambda: multiply_bonus(bonuses, '修炼速度倍率', 0.10),
        '紫气东来': lambda: (add_bonus(bonuses, '气运', 20), add_bonus(bonuses, '悟性', 10)),
        '草根崛起': lambda: (multiply_bonus(bonuses, '气血', 0.10), add_bonus(bonuses, '防御', 0.08)),
        '同舟共济': lambda: multiply_all_fetter_bonus(bonuses, 0.50),
    }
    
    effect_func = effect_map.get(fetter['name'])
    if effect_func:
        effect_func()
```

## 2.8 设计优势与创新点

### 2.8.1 刷初始机制设计

1. **随机性的乐趣**：每次随机都是新的开始，充满期待

3. **明确的目标**：玩家知道自己想要什么样的组合
4. **有限的重置**：广告和道具限制，防止无限刷

### 2.8.2 个性化设计

1. **灵根决定玩法倾向**：不同灵根适合不同成长路线
2. **命格决定战斗风格**：攻击型、防御型、平衡型等
3. **羁绊创造组合**：多种羁绊组合产生独特效果

### 2.8.3 长期影响设计

1. **灵根永久锁定**：初始选择影响整个游戏体验
2. **命格可优化**：通过命格石可以调整命格
3. **羁绊实时生效**：每次重置都重新计算羁绊
