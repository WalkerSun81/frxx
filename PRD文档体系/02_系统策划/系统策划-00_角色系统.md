## 0.3 角色成长体系

### 0.3.1 成长维度总览

```
角色成长
    │
    ├── 境界成长 (主线)
    │   └── 修为积累 → 境界突破 → 属性大幅提升
    │
    ├── 装备成长 (战力核心)
    │   └── 获取装备 → 强化/镶嵌 → 战力提升
    │
    ├── 灵根成长 (先天资质)
    │   └── 初始随机 → 影响修炼效率和元素伤害
    │
    ├── 命格成长 (个性化)
    │   └── 抽取命格 → 激活羁绊 → 属性加成
    │
    ├── 技能成长 (战斗手段)
    │   └── 境界解锁 → 升级强化 → 战斗效果
    │
    └── 生活技能 (资源产出)
        └── 采集/制作 → 材料/丹药 → 辅助成长
```

### 0.3.2 战力构成公式

```python
def calculate_combat_power(role_id):
    """
    计算角色总战力
    战力 = 境界基础 + 装备加成 + 灵根加成 + 命格加成 + 技能加成
    """
    role = get_role(role_id)
    attrs = get_attributes(role_id)
    
    # 境界基础战力 (占40%)
    realm_power = REALM_BASE_POWER[role.realm_level]
    
    # 装备战力 (占35%)
    equipments = get_equipped_items(role_id)
    equip_power = sum([calc_equip_power(e) for e in equipments])
    
    # 灵根战力 (占10%)
    element_root = json.loads(role.element_root)
    element_power = calc_element_power(element_root)
    
    # 命格战力 (占10%)
    talents = get_talents(role_id)
    talent_power = sum([t.power_bonus for t in talents])
    
    # 技能战力 (占5%)
    skills = get_skills(role_id)
    skill_power = sum([s.power_bonus for s in skills])
    
    total_power = (
        realm_power + 
        equip_power + 
        element_power + 
        talent_power + 
        skill_power
    )
    
    return total_power
```

### 0.3.3 各成长维度占比

| 成长维度 | 战力占比 | 成长速度 | 随机性 | 可替换性 |
|:---|:---:|:---:|:---:|:---:|
| **境界** | 40% | 慢(按天计) | 低 | 不可替换 |
| **装备** | 35% | 中(按小时计) | 高 | 可替换 |
| **灵根** | 10% | 极慢(一次性) | 极高 | 难重置 |
| **命格** | 10% | 慢(按天计) | 高 | 可重置 |
| **技能** | 5% | 中(按小时计) | 低 | 不可替换 |

### 0.5.2 最终属性计算公式

```python
def calculate_final_attributes(role_id):
    """
    计算角色最终属性
    所有加成均为乘算，避免数值膨胀
    """
    role = get_role(role_id)
    base_attrs = get_base_attributes(role.realm_level)
    
    # 获取各系统加成
    equip_bonus = get_equipment_bonus(role_id)      # 装备加成
    element_bonus = get_element_bonus(role_id)      # 灵根加成
    talent_bonus = get_talent_bonus(role_id)        # 命格加成
    skill_bonus = get_skill_bonus(role_id)          # 技能加成
    fetter_bonus = get_fetter_bonus(role_id)        # 羁绊加成
    
    final_attrs = {}
    for attr in ['strength', 'vitality', 'agility', 'intelligence', 
                 'accuracy', 'sense', 'wisdom', 'luck']:
        # 基础值
        base = base_attrs.get(attr, 10)
        
        # 各系统加成(乘算)
        equip_mult = equip_bonus.get(attr, 1.0)
        element_mult = element_bonus.get(attr, 1.0)
        talent_mult = talent_bonus.get(attr, 1.0)
        skill_mult = skill_bonus.get(attr, 1.0)
        fetter_mult = fetter_bonus.get(attr, 1.0)
        
        # 最终值 = 基础 × 装备 × 灵根 × 命格 × 技能 × 羁绊
        final = base * equip_mult * element_mult * talent_mult * skill_mult * fetter_mult
        final_attrs[attr] = int(final)
    
    return final_attrs
```

## 0.8 总结

### 0.8.1 角色系统核心职责

1. **数据容器**：汇聚所有子系统的数据，提供统一查询接口
2. **成长主线**：境界成长驱动游戏进度，解锁各系统功能
3. **战力计算**：整合各系统加成，计算最终战斗属性
4. **展示中心**：角色面板是玩家查看自身状态的核心入口

### 0.8.2 与其他系统的协作模式

```
角色系统 ←── 读取 ── 各子系统(提供数据)
    │
    └── 写入 ──► 各子系统(触发更新)
```

- **读取模式**：角色面板展示时，聚合查询各子系统数据
- **写入模式**：角色升级/突破时，触发各子系统的解锁/更新

### 0.8.3 后续扩展方向

| 扩展内容 | 说明 | 优先级 |
|:---|:---|:---:|
| 角色外观 | 时装、头像框、称号等外观系统 | P2 |
| 角色成就 | 成就系统，记录游戏里程碑 | P2 |
| 角色统计 | 详细的游戏数据统计 | P3 |
