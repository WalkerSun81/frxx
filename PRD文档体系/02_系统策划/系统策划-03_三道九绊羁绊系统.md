# 系统策划-03_三道九绊羁绊系统

---

## 1.1 系统概述

- **系统定位**：核心Build构筑层，决定角色发展方向
- **核心机制**："三选一×多组合"，每道激活1个羁绊
- **设计目标**：提供策略深度，满足不同流派玩家需求

---

## 1.2 羁绊体系架构

```
三道九绊
├── 天道层（全局规则，激活1个）
│   ├── 五行归一
│   ├── 五彩斑斓
│   └── 道法自然
│
├── 地道层（品质策略，激活1个）
│   ├── 紫气东来
│   ├── 草根崛起
│   └── 双子星辉
│
└── 人道层（属性专精，激活1个）
    ├── 战意凝形
    ├── 道法精修
    ├── 命运之子
    └── 不动如山
```

## 1.3 天道层羁绊详解

### 1.3.1 五行归一

| 项目 | 内容 |
|:---|:---|
| 激活条件 | 5颗天赋品质完全相同 |
| 激活概率 | 凡品流(40%)、良(30%)、优(15%)、极(10%)、仙(4%)、神(1%) |
| 效果 | 全属性+3% |
| 计算方式 | 按基础属性（不含装备/宝石）计算 |
| 适用流派 | 均衡型Build |

```python
def check_wuxing_guiyi(talents):
    qualities = [t['quality'] for t in talents]
    if len(set(qualities)) == 1 and len(qualities) == 5:
        return {
            'name': '五行归一',
            'activated': True,
            'bonuses': {
                '全属性百分比': 3
            }
        }
    return {'activated': False}
```

## 1.7 羁绊效果叠加规则

```python
def calculate_fetter_bonus(role):
    total_bonuses = {
        'strength_percent': 0,
        'vitality_percent': 0,
        'cultivate_speed': 0,
        'crit_rate': 0,
        'defense_percent': 0,
        'hp_percent': 0,
        'all_attr_percent': 0,
        'wisdom_percent': 0,
        'sense_percent': 0,
        'luck_percent': 0,
    }
    
    activated_fetters = get_activated_fetters(role.talents)
    
    for fetter in activated_fetters:
        if fetter == '五行归一':
            total_bonuses['all_attr_percent'] += 3
        elif fetter == '五彩斑斓':
            total_bonuses['sense_percent'] += 8
            total_bonuses['wisdom_percent'] += 6
        elif fetter == '道法自然':
            total_bonuses['cultivate_speed'] += 10  # 百分比
        elif fetter == '紫气东来':
            total_bonuses['all_attr_percent'] += 2
            total_bonuses['crit_rate'] += 5
        elif fetter == '草根崛起':
            total_bonuses['hp_percent'] += 10
            total_bonuses['defense_percent'] += 8
        elif fetter == '双子星辉':
            # 需要单独计算，见下方
            apply_duplicate_bonus(total_bonuses, role.talents)
        elif fetter == '战意凝形':
            total_bonuses['crit_rate'] += 5
            total_bonuses['strength_percent'] += 8
        elif fetter == '道法精修':
            total_bonuses['cultivate_speed'] += 8
            total_bonuses['wisdom_percent'] += 6
        elif fetter == '命运之子':
            total_bonuses['luck_percent'] += 8
        elif fetter == '不动如山':
            total_bonuses['hp_percent'] += 8
            total_bonuses['defense_percent'] += 6
    
    return total_bonuses
```

## 1.8 服务端实现要点

### 1.8.1 羁绊计算时机

- 角色登录时计算一次
- 命格重置后重新计算
- 属性面板请求时返回当前羁绊状态

### 1.8.2 羁绊数值应用

```python
def calculate_final_attributes(role):
    # 1. 基础属性（境界+天赋基础值）
    base = get_base_attributes(role)
    
    # 2. 羁绊加成
    fetter_bonus = calculate_fetter_bonus(role)
    
    # 3. 应用百分比加成
    for attr, percent in fetter_bonus.items():
        if 'percent' in attr:
            base[attr.replace('_percent', '')] *= (1 + percent / 100)
    
    # 4. 装备/宝石等绝对值加成
    equipment_bonus = get_equipment_bonus(role)
    base = add_values(base, equipment_bonus)
    
    return base
```
