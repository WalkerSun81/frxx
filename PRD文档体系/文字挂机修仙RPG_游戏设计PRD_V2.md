## 一、文档概述

### 1.1 项目定位

| 维度 | 说明 |
|:---|:---|
| **游戏名称** | 《凡人修仙录》（暂定） |
| **核心品类** | 文字挂机修仙RPG（Idle Cultivation RPG） |
| **平台** | Web端（HTML5浏览器），移动端自适应，竖屏设计 |
| **商业模式** | 免费下载 + 内购（道具/加速/抽卡） |
| **目标用户** | 25-35岁，喜爱修仙文化、接受长期数值养成的玩家 |
| **核心体验** | 主动推图验证Build + 推图驱动挂机效率 + 境界突破形成阶段循环 |

### 1.2 设计理念

> **核心循环权威定义**：战斗推图是玩家上线后的主要主动玩法；挂机修炼是主要长期修为来源；最高通关层决定挂机效率，Boss提供突破材料。修炼按钮只用于有限的短时闭关，不能成为无限点击的最优成长方式。

1. **更深度的Build系统** — 三道九绊策略层 + 装备/宝石/技能搭配
2. **更透明的数值反馈** — 公式完全公开，满足硬核玩家计算需求
3. **更丰富的养成维度** — 11+养成系统，支撑长期留存
4. **更沉浸的叙事体验** — 古风文案包装，增强代入感

### 2.3 属性影响全链路

以**气运**为例，展示一个属性如何贯穿全系统：

```

  ├── 战斗系统 → 暴击伤害加成（渐近增长）
  ├── 经济系统 → 灵石收益倍率（气运×1.0X）
  ├── 掉落系统 → 装备/材料掉落率+（渐近增长，极限+80%）

  ├── 命格系统 → "命运之子"羁绊激活条件
  └── 社交系统 → 宗门贡献加成（间接）
```

## 四、全局经济循环模型

### 4.1 货币体系

| 货币 | 获取途径 | 消耗途径 | 通胀控制 |
|:---|:---|:---|:---|
| **灵石**（硬通货） | 打怪掉落、任务奖励、宗门俸禄 | 装备强化、坊市购买、突破材料 | 境界越高消耗越大（指数级） |
| **铜板**（软通货） | 打怪掉落、出售装备、生活技能 | 基础强化、购买消耗品 | 自然消耗（强化/购买） |
| **修为**（经验值） | 挂机修炼、战斗奖励、丹药 | 境界突破消耗 | 突破需求指数级增长 |
| **宗门贡献** | 宗门任务、宗门战 | 宗门商店购买 | 每日上限 |
| **命格石**（稀有） | 日常任务、活动、付费 | 重置命格 | 严格限量 |

### 4.2 灵石经济平衡表（每日预估）

| 阶段 | 日产出（免费） | 日消耗（必要） | 日结余 | 累积周期 |
|:---|:---:|:---:|:---:|:---|
| 练气期 | ~500 | ~200（强化+突破材料） | +300 | 3天突破 |
| 筑基期 | ~1,200 | ~800（强化+丹药+材料） | +400 | 5天突破 |
| 金丹期 | ~2,500 | ~2,000（法宝+高级强化） | +500 | 7天突破 |
| 元婴期 | ~5,000 | ~4,200（多法宝+高级材料） | +800 | 10天突破 |
| 化神期 | ~10,000 | ~9,000（全系统消耗） | +1,000 | 14天突破 |
| 炼虚期+ | ~20,000+ | ~18,000+ | +2,000+ | 20天+/境界 |

**设计原则**：
- 日结余始终为正，保证免费玩家可推进
- 高境界消耗增速 > 产出增速，自然引导付费
- 突破材料为阶段性大量消耗点（"卡点"设计）

### 4.3 铜板经济平衡表（每日预估）

| 阶段 | 日产出 | 日消耗 | 说明 |
|:---|:---:|:---:|:---|
| 练气期 | ~300 | ~150 | 基础强化消耗 |
| 筑基期 | ~600 | ~400 | 强化频率增加 |
| 金丹期 | ~1,200 | ~900 | 多装备同时强化 |
| 元婴期 | ~2,000 | ~1,600 | 法宝强化加入 |
| 化神期+ | ~3,000+ | ~2,500+ | 基本持平，引导出售装备补铜板 |

### 4.4 通胀控制机制

| 机制 | 说明 |
|:---|:---|
| **消耗水槽** | 装备强化（无限消耗）、坊市购买（每日刷新）、突破材料（阶段性大额） |
| **产出上限** | 挂机收益有8小时上限、每日任务有次数限制 |
| **货币回收** | 高境界突破消耗指数级增长、高级强化消耗剧增 |
| **时间门槛** | 所有高价值产出都需要时间积累，无法跳过 |

## 七、战斗数值体系（细化）

### 7.1 伤害计算完整公式

```
# 最终伤害
最终伤害 = 基础伤害 × 暴击系数 × 技能系数 × 羁绊系数 × 层数系数

# 基础伤害
基础伤害 = (攻击方ATK - 防御方DEF × 0.5) × 随机浮动(0.9~1.1)
基础伤害 = max(基础伤害, 攻击方ATK × 0.1)  # 最低保底

# 暴击系数
if 暴击判定成功:
    暴击系数 = 1.5 + 气运 × 0.02  # 基础150%，气运每点+2%
else:
    暴击系数 = 1.0

# 技能系数
技能系数 = 技能基础倍率 × 功法加成

# 防御减免（收益递减）
实际减伤 = DEF / (DEF + 500)
```

### 7.3 战斗平衡验证

**验证场景**：练气圆满玩家（战力620）vs 青云村外第20层妖狐（HP 600, ATK 65, DEF 35）

```
玩家攻击 = 53(力量) + 30(白板武器) = 83
基础伤害 = (83 - 35×0.5) × 1.0 = 65.5
怪物减伤 = 35/(35+500) = 6.5%
实际伤害 = 65.5 × (1-0.065) = 61.2

击杀回合 = 600 / 61.2 ≈ 10回合
怪物对玩家伤害 = (65 - 93×0.3×0.5) × 0.9 = 65 - 14 = 51 × 0.9 = 45.9
10回合受到总伤害 = 45.9 × 10 = 459

玩家气血 = 780
生存判定 = 780 > 459 → ✅ 可通关（剩余气血约40%）

挂机伤害/秒 = 29/8 + 53/10 = 9
挂机击杀时间 = 600 / 9 = 67秒 ≈ 1分钟/只
```

**结论**：练气圆满玩家可以挂机通过青云村外第20层，符合预期。

### 7.4 挂机推图数值模型

```
# 挂机通关条件
挂机DPS × 预期击杀时间 > 怪物HP
且 怪物DPS × 预期击杀时间 < 玩家HP × 0.7（留30%安全余量）

# 推图效率
每只怪耗时 = 怪物HP / 挂机DPS
每层耗时 = 怪物数量 × 每只耗时
每层收益 = 怪物数量 × (灵石 + 材料期望值)
每小时收益 = 3600 / 每层耗时 × 每层收益
```

### 11.4 挂机收益模型

```
# 挂机推图判定
可挂机最高层 = f(挂机DPS, 玩家HP, 怪物DPS)
条件: 挂机击杀时间 < 30秒 且 生存率 > 90%

# 离线收益计算
离线时长 = min(实际离线时长, 8小时)  # 上限8小时
离线修为 = 修炼速度 × 离线时长 × 0.8  # 80%效率
离线灵石 = 当前地图每分钟灵石 × 离线时长 × 0.5  # 50%效率
离线材料 = 当前地图每分钟材料期望 × 离线时长 × 0.3  # 30%效率

# 示例：筑基圆满玩家离线8小时
修炼速度 = ~250/s
离线修为 = 250 × 28800 × 0.8 = 5,760,000修为
离线灵石 = ~2/分钟 × 480分钟 × 0.5 = 480灵石
离线材料 = ~0.3个/分钟 × 480 × 0.3 = 43个材料
```
